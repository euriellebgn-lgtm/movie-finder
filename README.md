# movie-finder
FindAnyMovie is a simple, interactive web application that allows users to search for movies, view trending films, and see detailed information about each movie including release date, runtime, genres, and where to watch them. The project leverages The Movie Database (TMDb) API to fetch movie data dynamically.

Features
-Trending Movies: Automatically loads the most popular movies of the day on the homepage.

-Live Search: Search for any movie in the TMDB database with instant results.

-Detailed View: Click on any movie card to see:

-Plot overview and release date.

-Runtime and Genre tags.

-Average user ratings.

-Watch Providers: Dynamically fetches streaming availability.

-Responsive Design: A grid-based layout that adapts to different screen sizes.

Built With
-Frontend: HTML5, CSS3 (CSS Grid)

-Logic: JavaScript (Fetch API, Async/Await)

-API: The Movie Database (TMDB)


Usage
Home: On page load, you'll see the current trending movies.

Searching: Type a title into the search bar and press Enter or click the 🔎 button.

Details: Click on a movie poster or title. The grid will update to show the specific details and available streaming platforms for that movie.

Reset: Click the "Movie Finder 🎬" title at any time to return to the home screen.
